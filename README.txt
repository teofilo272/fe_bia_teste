# Fe & Bia ❤️ - App Romântico

## Como Rodar no seu Computador:

1. Instale o Python: https://www.python.org/downloads/
2. No terminal, navegue até a pasta do projeto.
3. Instale as dependências com:
   pip install -r requirements.txt
4. Rode o app com:
   streamlit run app.py
5. O app abrirá no seu navegador (http://localhost:8501).

## Como Publicar Online (gratuito!):

1. Crie uma conta no GitHub (https://github.com) e envie esses arquivos para lá.
2. Crie uma conta no Streamlit Cloud (https://streamlit.io/cloud) e conecte com seu GitHub.
3. Clique em "New App", selecione o repositório e clique em Deploy.

O Streamlit criará um link online para você enviar para quem quiser ou transformar em QR Code.

## Personalização:

- Edite o arquivo `app.py` para alterar perguntas, respostas, mensagens ou o link da música.
- O campo `link_musica` na função `tela_final()` é onde você pode mudar o link da música.
